About This Tracker

This is a personal, secure financial ledger engineered to manage daily income, track expenses, and monitor debt repayment in RWF. It connects to a cloud database so data syncs across devices, but all numbers are AES-encrypted locally before being sent to Firebase, ensuring complete financial privacy.
